Debate Timer

Debate Timer, or CX Timer, is a desktop timer made for High School debate competitions. The timer simplifies debate rounds by offering the user a variety of useful functions. The timer is being used by many debaters in the high school community. This project is written in Java (using various Java 8 principles, including Lambdas) with Oracle's JavaFX pacakages to create the GUI.
